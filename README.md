# analog-clock
Vue.js로 구현한 아날로그 시계입니다.
다음과 같은 기능을 구현했습니다.

- 현재 시간을 기준으로 한 시침, 분침, 초침 표시
- 시계에 마우스 오버시 현재 시간을 툴팁으로 표시
